Code examples for Polycom KWS XML-RPC

This is a number of XML-RPC samples used to demonstrate the use of the XML-RPC API for the Polycom KWS products - currently KWS300 and KWS6000.

One sample is written in Java, one in Visual Basic .NET and the rest in Python. The samples are simple and easy to understand even if you do not know the Python language. This README explains how to get started with the examples.

Java example

This is a sample illustrating how to access the Polycom KWS XML-RPC interface from Java. The sample uses Apache RPC-XML package for the XML-RPC client. The package is included and can be obtained from http://ws.apache.org/xmlrpc/.

Executing the example

The samples are tested with OpenJDK java build 1.6.0_0-b14 and Sun java build 1.6.0_07-b06.

To use the sample modify SendSMS.java to match your test system, compile and execute the command:

Linux

./sendsms.sh

Microsoft Windows

sendsms.bat

Visual Basic .NET example

This is a sample illustrating how to access the Polycom KWS XML-RPC interface from Visual Basic .NET. The sample uses the Cook Computing RPC-XML library for the XML-RPC client. The library is included and can be obtained from http://www.xml-rpc.net/. The sample is created using Microsoft Visual Studio Express 2010.

Python examples

There is a number of Python XML-RPC samples used to demonstrate the use of the XML-RPC API for the Polycom KWS products - currently KWS300 and KWS6000.

Obtaining Python

The samples are tested with Python version 2.7. Please be aware that Python 3.x is a lot different from Python 2.x and the samples will not work with Python 3.x.

Linux

Most Linux distributions are shipped with Python installed. If Python is not installed use your package manager to install Python.

Microsoft Windows

Download Python from http://www.python.org and install.

Executing the examples

To use the samples modify the files to match your test system and execute the command:

python <sample-file> <KWS address> <to extension>

For example:

python sendsms.py 192.168.0.1 2601

Examples

python/broadcast.py Broadcast a message to all handsets.
python/game_token.py A simple game demonstrating the features of the 6040 and 7040 handsets.
python/hardwareextension.py Access the hardware extensions of the 6040 and 7040 handsets.
python/introspection.py Get information about the methods provided by the XML-RPC interface.
python/makecalladvanced.py Make a text call using advanced features.
python/makecallmenu.py Make a text call and display a menu in the handset.
python/makecallprompt.py Make a text call and display a prompt in the handset to receive input from the user.
python/makecall.py Make a basic text call.
python/receivecall.py Receive a text call initiated from a handset.
python/receivesms.py Receive a SMS send from a handset.
python/redundancy.py Demonstrates how to create an application with failover for KWS6000 redundancy.
python/requestbasestations.py Request the list of base stations from a handset. Used for a basic location service.
python/requeststatus.py Request status information from a handset.
python/sendsmshttps.py Send a SMS securely to a handset using XML-RPC via HTTPS.
python/sendsms.py Send a SMS to a handset.
python/smsadvanced.py Send an advanced SMS to a 6040 or 7040 handset.
python/smsrelay.py SMS relay that will receive an SMS and send it to the requested handset.
python/status.py Receive status information from handsets.
